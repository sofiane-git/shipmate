# Design history (archive)

`docs/specs/` and `docs/plans/` are **frozen archives** of how shipmate v0 was designed and
built — the *why* and the *how-we-built-it*. They are **not** kept in sync with the code.

**For current behavior, read the living docs instead:**
- [`README.md`](../README.md) — what shipmate is and does
- [`docs/quickstart.md`](quickstart.md) — using it
- [`docs/positioning.md`](positioning.md) — when (not) to use it
- [`CHANGELOG.md`](../CHANGELOG.md) — what changed
- `skills/*/SKILL.md` + `scripts/*.sh` + `tests/` — the actual behavior

If a frozen doc and the code disagree, **the code wins**. New design work gets a new dated
note, never an edit to a frozen one.
